============
Installation
============


Stable release
--------------

To install Stock Price Valuation Predict, run this command in your terminal:

.. code-block:: console

    $ pip install predict_stock_price_valuations

This is the preferred method to install Stock Price Valuation Predict, as it will always install the most recent stable release.

If you don't have `pip`_ installed, this `Python installation guide`_ can guide
you through the process.

.. _pip: https://pip.pypa.io
.. _Python installation guide: http://docs.python-guide.org/en/latest/starting/installation/


From sources
------------

The sources for Stock Price Valuation Predict can be downloaded from the `Github repo`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/joaothomazlemos/predict_stock_price_valuations

Or download the `tarball`_:

.. code-block:: console

    $ curl -OJL https://github.com/joaothomazlemos/predict_stock_price_valuations/tarball/master

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python setup.py install


.. _Github repo: https://github.com/joaothomazlemos/predict_stock_price_valuations
.. _tarball: https://github.com/joaothomazlemos/predict_stock_price_valuations/tarball/master
